# 对话框（Dialogs）清单（MVP）

## P0（必须）
- Open Case Folder：选择包含 `request.json + mesh.npz` 的目录
- Select Solver：输入 `fake` 或 `python:<module>`
- Run Solve：运行确认（展示 pre-check 结果，后续增强）
- Solver Log/Errors：失败时展示错误码/堆栈（若有）与导出诊断包按钮
- About：版本信息、契约版本、python 版本

## P1（建议尽早）
- Preferences：单位显示、主题、渲染设置
- Export Results：导出 VTK/CSV/图片
